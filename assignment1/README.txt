1. 과제 설명
	비트 연산 과제 입니다. 
	코드 작업과 리포트 작업이 있습니다.
	코드 작업은 비트 연산과 관련된 10개의 빈 펑션들이 bits.c파일 안에 제공 되어지며 
	빈펑션들을 주석에서 설명하는 기능으로 작동하도록 코드를 채워주시면 됩니다.  
	리포트 작업은 각각의 펑션들을 어떤식으로 구현하였는지 1-2페이지로 작성하여 주시면됩니다. 
	제출하신 페이지수가 두장보다 많을시 점수에 불이익이 있습니다.

	코드는 Gitlab에 2020/10/6 11:59PM 까지 올려주시기 바립니다.
	(gitLab에 업로드가 가능한지 미리 확인 부탁드립니다.)
	코드를 올릴때는 assignment1/bits.c 형태로 업로드 하시면 됩니다. (assignment1/bits.c 파일만 제출)
	bits.c 코드는 ./driver.pl이 에러없이 정상적으로 돌아가는것을 확인한후 제출하셔야 합니다. 
	제출이 이와 다를시 점수에 불이익이 있을수 있습니다.
	
	리포트는 프린트하여 2020/10/7 실습시작 전까지 제출하시기 바랍니다. 
	24시간까지 늦게 제출하는 경우에 50% 감점이며 24시간 이후 제출은 0점입니다.

2. 과제 수행 및 테스트 방법
	2.1 datalab-handout 디렉터리로 이동
	2.2 bits.c 파일을 에디터로 오픈 
	2.3 비어있는 함수의 내용을 수정(함수 위의 주석을 참고) 후 저장
	2.4 ./driver.pl bits.c 실행
	2.5 실행 결과를 확인 
	
	주의사항 : 각 함수별 사용 가능한 연산자만 사용 해야 함. (연산자 개수 제한 있음)

	
3. driver.pl 실행 시 에러가 발생하는 경우 
	- 라이브러리 설치
		sudo apt-get install g++-multilib
		sudo apt-get install make

		위의 패키지 설치후 에러가 발생 시 
			sudo apt-get update
			sudo apt-get upgrade
			sudo apt-get install g++-multilib
			sudo apt-get install make
